## Key Concepts to Master

- **Isaac Sim**: NVIDIA's high-fidelity simulation platform for robotics development
- **Isaac ROS**: Collection of robotics libraries and tools for ROS/ROS2
- **Visual SLAM (VSLAM)**: Technology enabling robots to map environments and determine position using visual sensors
- **Sensor Pipelines**: Data processing chains that transform raw sensor data into actionable information
- **Nav2**: ROS 2 navigation stack for path planning and execution
- **Simulation-to-Real Transfer**: Techniques for deploying AI behaviors from simulation to real robots
